#Taller de Análisis de datos y programación con R

##Objetivo: 

Enseñar  a los participantes las bases del lenguaje de programación y estadísticas en R con el fin de darles herramientas para la manipulación de datos, generación de análisis estadísticos, programación y representación gráfica.

##Expectativas: 

Al final del curso se espera que los participantes tengan habilidades en R que les permitan analizar  sus datos y generar resultados para  tomar decisiones.
